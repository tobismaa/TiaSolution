Role Features

Features Available to Each Role

Business Admin

  • Manage users and access rights
  • Manage branches and business structure
  • Oversee payroll setup
  • Monitor business-wide finance controls

Head of Operations

  • Review pending branch activity
  • Approve invoices and expenses
  • Monitor due values and spend watch
  • View branch-level operational reports

Finance

  • Access payroll-linked finance workflows
  • Open general ledgers and chart of accounts
  • Generate trial balance
  • Run financial reports and validations

Operations

  • Capture expenses and branch transactions
  • Post GL activity
  • Track posting volume and pending approvals
  • Work within active branch scope

Auditor

  • Read-only access to trial balance and reports
  • Inspect ledgers and audit trail
  • Validate balances and control evidence
  • Review without editing live data

Access Principle

Each role sees only the functions required for its operational responsibility, reducing clutter and protecting sensitive records.

Client Value

This role-based model improves control, speeds adoption, and makes responsibilities easier to understand across teams.
